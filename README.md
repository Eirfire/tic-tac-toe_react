## Getting started
## Clone from git hub:
```
git clone https://github.com/Eirfire/tic-tac-toe_react.git
cd tic-tac-toe_react
```
## Npm -
```
npm install
npm start
``` 
## Yarn -
```
yarn install
yarn serve
```
